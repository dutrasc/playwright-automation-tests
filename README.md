# 🎭 Playwright Automation Tests

Automated E2E test suite built with Playwright and TypeScript, covering multi-browser testing across Chromium, Firefox and WebKit.

## 🚀 Tech Stack

- [Playwright](https://playwright.dev/) — E2E test framework
- TypeScript — strongly typed test code
- GitHub Actions — CI/CD pipeline

## 📁 Project Structure

```
tests/
└── example.spec.ts   # E2E test scenarios
```

## ▶️ How to Run

```bash
# Install dependencies
npm install

# Run all tests
npx playwright test

# Run with UI mode
npx playwright test --ui

# View HTML report
npx playwright show-report
```

## 🌐 Browsers

Tests run on:
- Chromium
- Firefox
- WebKit
